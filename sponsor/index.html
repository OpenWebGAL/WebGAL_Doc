                                              # 赞助

                                              WebGAL 是一款开源软件，因此你可以免费在 MPL-2.0 开源协议的范畴下使用本软件，并可用于商业使用。

                                              但即便如此，你的赞助也可以给予开发者前进的动力，让这个项目变得更好。

                                              # 展示赞助信息

                                              无论你使用何种形式赞助，你都可以在 WebGAL 的 GitHub 项目主页和网站展示您的信息（个人主页、公司主页、GitHub 资料页等）。如需展示，请在留言中留下需要展示的内容或将内容连同赞助收据发送至 contact@openwebgal.com
