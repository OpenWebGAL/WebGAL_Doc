                                            # 新特性预告

                                            WebGAL 的开发计划基本上是用 issues 跟踪的，请访问 WebGAL issuesopen in new window 或 WebGAL Terre issuesopen in new window 来查看。

                                            # 特性请求与缺陷报告

                                            如果你想要请求任何 WebGAL 引擎以及编辑器的新特性，或报告任何缺陷，请使用下文所述的 Issues 页面或发送邮件来向开发者报告。Issues 和邮件更加正式并且会被长期保存，因此需要通过这种方式来记录。
