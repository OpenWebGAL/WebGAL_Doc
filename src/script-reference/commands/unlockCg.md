# unlockCg

将图片收录进 CG 鉴赏。

## 语句内容

填写图片的路径。

```webgal
unlockCg:chapter_03/date.png -name=第一次约会;
```

## 参数

### name
<!-- @include: ../arguments/name-cg.md -->
若不填写 `name`，则以图片的路径作为名称。

```webgal
unlockCg:chapter_03/date.png -name=第一次约会;
```

### series
<!-- @include: ../arguments/series-cg.md -->
```webgal
unlockCg:chapter_03/date.png -name=第一次约会 -series=角色B;
```
